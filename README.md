# Zombanga-Assets-OS-Game-Project-
Zombanga is an open-source 2D tower defense game that gamifies core Operating System concepts by casting the player in the role of the system kernel. Instead of just reading about CPU scheduling, players must actively manage a system economy, handle strict tower resource limits, and deploy defensive nodes utilizing real-world algorithms like FCFS, SJF, and PPS to eliminate incoming "malware" zombies. Set against a nostalgic, retro system diagnostic visual style, the game features a uniquely localized setting with a custom pixel art map modeled after the Plaza in Calamba City, Laguna. This repository contains the complete, original creative asset pipeline for the project, showcasing my custom pixel art, character sprites, environment designs, and frame-by-frame animation sheets for all towers and enemies.
